# Stock Portfolio Application
A Java application that allows the user to withdraw and deposit cash from the users brokerage account and the ability to purchase and sell stock.

+ Deposit Cash
+ Withdraw Cash
+ Buy Stock
+ Sell stock
+ Display Transaction History
+ Display Portfolio Information
